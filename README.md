# Серверное приложение информационной системы NeuroNotes 

Данный репозиторий содержит исходный код серверной части информационной системы интеллектуальной обработки аудио-текстовых данных, разработанной в рамках выпускной квалификационной работы. Исходный код клиентского веб-приложения представлен в связанном репозитории [neuro-notes-web-client](https://github.com/Fairlier/neuro-notes-web-client). Программный комплекс автоматизирует цикл работы с персональными записями. Разработанная система выполняет преобразование речи в текст, автоматическую тематическую классификацию заметок, формирование их структурированного представления и краткого резюме, а также осуществляет семантический и диалоговый доступ к информации с применением механизма генерации ответов с дополненной выборкой (RAG). 

Архитектура серверного приложения спроектирована в соответствии с принципами предметно-ориентированного проектирования (DDD) и чистой архитектуры. В основе программного интерфейса лежит принцип разделения ответственности команд и запросов (CQRS). Реализация выполнена на базе платформы ASP.NET Core 8. В качестве основной системы управления базами данных задействована PostgreSQL с векторным расширением pgvector. Для хранения бинарных аудиофайлов интегрировано локальное объектное хранилище MinIO. Выполнение ресурсоемких задач, включая обращения к нейросетевым моделям и транскрибацию, вынесено в подсистему фоновой асинхронной обработки на базе библиотеки Hangfire. 

Интеллектуальная обработка данных построена на гибридном подходе с использованием локальных и облачных моделей машинного обучения. Локальная транскрибация речи реализована посредством акустических моделей Vosk. Тематическая классификация текстов выполняется с помощью дообученной дистиллированной многоязычной модели семейства BERT, экспортированной в формат ONNX и запускаемой через среду ONNX Runtime. Взаимодействие с большими языковыми моделями (LLM) абстрагировано и позволяет подключать локальные генеративные модели через платформу Ollama, а также внешние коммерческие программные интерфейсы Gemini и Mistral.
